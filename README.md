# Todo Application

Create a simple Todo application that stores tasks in in-memory storage (In array).

### Setup Instructions

#### 1. Backend Setup

1. Navigate to the backend folder:
   ```bash
   cd /backend
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

#### 2. Frontend Setup

1. Navigate to the frontend folder:
   ```bash
   cd /frontend
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```
